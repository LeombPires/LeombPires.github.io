3D Renderer
3D Graphics · JavaScript · p5.js

A from-scratch 3D rendering pipeline built without a game engine or 3D library. Implements perspective projection, real-time rotation, back-face culling, painter's algorithm depth sorting, and per-face Lambertian shading via dot-product lighting. Supports multiple mesh types and includes an interactive light-direction widget — all UI drawn directly on the canvas.

Perspective Projection Back-face Culling Painter's Algorithm Multiple Mesh Types Dot Product Lighting Interactive Light Direction
◆ Side Quest
Text Particle System
Interactive Simulation · JavaScript · p5.js

An interactive particle system that converts text into hundreds of individual points, each with its own physics. Supports two modes: a magnifying glass that reveals hidden text by easing nearby particles toward their letter positions, and a vortex repulsion mode where arranged text scatters when the cursor approaches — demonstrating spatial queries, force-based movement, and target-seeking behaviour.

Particle System Force-based Physics Spatial Interaction Input-driven Behaviour
