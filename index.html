◆ Side Quest
3D Cube Renderer
3D Graphics · JavaScript · p5.js

A from-scratch 3D rendering exercise built without a game engine or graphics library. Implements perspective projection, real-time rotation, back-face culling to hide non-visible faces, and per-face lighting calculated via the dot product between face normals and a directional light vector. Includes an interactive widget to adjust the light direction in real time.

Perspective Projection Back-face Culling Dot Product Lighting Interactive Light Direction
◆ Side Quest
Text Particle System
Interactive Simulation · JavaScript · p5.js

An interactive particle system that converts text into hundreds of individual points, each with its own physics. Supports two modes: a magnifying glass that reveals hidden text by snapping nearby particles into their letter positions, and a vortex repulsion mode where arranged text scatters when the cursor approaches — demonstrating spatial queries, force-based movement, and target-seeking behaviour.

Particle System Force-based Physics Spatial Interaction Input-driven Behaviour
